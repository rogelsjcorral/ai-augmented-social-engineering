# AI-Augmented Social Engineering

Technical position paper on AI-augmented social engineering as identity-to-control-plane risk.

## Summary

This repository contains a technical position paper arguing that AI-augmented social engineering should not be treated merely as better phishing content. In modern environments, a trust error can become an identity event, and an identity event can become a control-plane event when privilege boundaries, delegation hygiene, recovery workflows, and detection are weak.

The paper introduces a vendor-neutral threat model centered on recursive trust and proposes practical defensive objectives for reducing identity-to-control-plane escalation risk.

This repository also includes a companion tabletop exercise designed to help organizations rehearse and evaluate identity-to-control-plane escalation scenarios in a structured way.
